# Documentation

This `docs/` folder contains extended documentation for the bot beyond the top-level `README.md`.

Contents:

- `setup.md` — Detailed setup and deployment instructions.
- `cogs.md` — How to write and register cogs, with a template and examples.
- `data-format.md` — Describes JSON data files and sample structures (e.g., `data/ranks.json`).
- `validation.md` — Explanation of `validate_bot.py` and how to interpret results.

Use these pages as the source of truth for development and contributions.
