# Code of Conduct

This project follows a standard Contributor Covenant style code of conduct. Be respectful, kind, and constructive.

Expected behavior:

- Be welcoming to newcomers.
- Assume good faith during discussions.
- Use inclusive language.

Unacceptable behavior:

- Harassment, hate speech, or threats.
- Personal attacks, doxxing, or targeted abuse.

If you experience or witness unacceptable behavior, open an issue or contact the repository owner privately.
